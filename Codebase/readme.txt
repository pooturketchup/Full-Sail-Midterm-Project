After creating the strawman application copy the project files to this location.

Set up Unity to work well with Git

1. Open the skeleton Unity project
2. Open the Editor setting under Project Settings found on the Edit menu.
3. Set Version Control mode to “Visible Meta Files”.
4. Set Asset Serialization Mode to “Force Text”.
5. Save and then close the project.

Commit and push the build to the remote repo

1. Right flick on the folder and select to add files using the Git interface
2. Commit the build and push it to the remote repo with a comment defining it as the first project commit
